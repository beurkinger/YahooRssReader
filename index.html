    A propos

  • Cette page est un petit lecteur de flux RSS dédié à Yahoo Actualités. Elle vise à récupérer les fils d'information de Yahoo et à les mettre en forme d'une manière agréable (enfin, je l'espère).
  • Ce petit projet a constitué pour moi une manière de me familiariser avec Javascript, et d'aborder plusieurs technologies Ajax (DOM, XMLHttpRequest, XML, etc.).
  • D'un point de vue technique, le script commence par récupérer le fichier XML du flux demandé auprès d'une page PHP dédiée. Il parcourt ensuite le DOM du fichier afin d'en extraire les informations pertinentes (à l'aide de regex, si nécessaire), qu'il réinjecte ensuite au sein du DOM de la présente page.
